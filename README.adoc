= Garland

This repository contains all my NixOS configurations.

== Configurations

All my configurations are named after poisonous plants, not because I am toxic, but because I think they are cool.

* **Wisteria**: my desktop computer 
* **Foxglove**: my framework laptop
* **Hemlock**: my VPS machine
* **Ivy**: Home Assistant running on Raspberry PI

== Secrets management

Secrets are managed with https://github.com/Mic92/sops-nix[sops-nix]. Encrypted secrets are stored in `secrets/<hostname>.yaml` and decrypted at NixOS activation time using the host's SSH ed25519 key.

=== First-time setup

Export the age private key from the SSH host key (once per machine):

[source,bash]
----
mkdir -p ~/.config/sops/age
sudo nix run nixpkgs#ssh-to-age -- -private-key -i /etc/ssh/ssh_host_ed25519_key > ~/.config/sops/age/keys.txt
----

=== Editing secrets

[source,bash]
----
just secrets <hostname>
----

This opens the decrypted file in your editor. On save, sops re-encrypts automatically.

=== Adding a new host

1. Get the age public key from the host's SSH key:
+
[source,bash]
----
# From the host itself:
cat /etc/ssh/ssh_host_ed25519_key.pub | nix run nixpkgs#ssh-to-age

# Or remotely:
ssh-keyscan -t ed25519 <host> | nix run nixpkgs#ssh-to-age
----

2. Add the key to `.sops.yaml` under `keys:` with an anchor (e.g. `&hostname`), and add a `creation_rules` entry for `secrets/<hostname>.yaml`

3. Create the encrypted secrets file:
+
[source,bash]
----
just secrets <hostname>
----
